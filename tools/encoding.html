Engineering Notes

  • Base64: Converts binary data to ASCII. Adds ~33% overhead. Used for email attachments and embedding images (Data URIs).
  • URL Encoding: Escapes unsafe ASCII characters (like space to %20) for HTTP requests.
  • HTML Entities: Converts reserved characters (`<` to `<`) to prevent XSS attacks and rendering issues.
  • Hex (Base16): Represents binary data using 0-9 and A-F. Each byte is exactly 2 characters.
  • File Loader: Uses the browser's `FileReader` API to convert local files directly into Base64 Data Strings. No data leaves your device.
