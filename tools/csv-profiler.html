Parsing Controls

Changing either setting reparses the current file locally.

Limits

  • Files above 20 MB are blocked before parsing.
  • Profiling stops after 200,000 non-header data rows.
  • Logical rows above 250,000 characters are blocked.
  • Files above 512 columns are blocked.
  • Distinct counts are exact up to 20,000 values per column.
  • Histograms use sampled numeric values to limit memory use.

Use "Load Good Sample" for a clean baseline. Use "Load Bad Sample" to exercise mixed comma, tab, semicolon, and pipe delimiters, ragged rows, duplicate rows, missing values, duplicate headers, and malformed quoting. The sample buttons preload comma delimiter and header-row mode.
