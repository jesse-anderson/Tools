Lamport Timestamps

A distributed clock algorithm for ordering events across multiple processes. Each process maintains a local counter that increments on:

  • Local events: Increment clock by 1
  • Send messages: Increment clock, include timestamp in message
  • Receive messages: Update clock to max(local, received), increment

Vector Clocks

Extend Lamport timestamps with process IDs. Each process has a vector of counters - useful for distinguishing concurrent events.
