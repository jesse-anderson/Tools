How To Use This

  1. Write the formula using variable names and standard operators.
  2. Set each variable's value and its uncertainty. Pick absolute, percent, or exact (a constant with no error).
  3. Read the result as value ± 1σ, then raise the coverage factor k for a wider confidence interval.
  4. Expand the steps to see the partial derivatives and which variable drives the error.

The Math

For independent variables, first-order (Taylor) propagation gives

σ_f = √( Σ (∂f/∂xᵢ · σᵢ)² )

Each term ∂f/∂xᵢ · σᵢ is that variable's share of the total error, added in quadrature.

Scope & Limits

Estimates only - verify before you rely on them

This uses first-order propagation and assumes variables are independent (uncorrelated). It is accurate when uncertainties are small relative to the values, so the formula is roughly linear across ±σ.

For strongly nonlinear formulas or large uncertainties, enable the Monte Carlo check. When it disagrees with the Taylor result, trust the sampled spread. Correlated inputs are not modeled.
