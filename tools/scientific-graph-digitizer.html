Scientific Graph Compare

// SIMPLE PAPER FIGURE MEASUREMENT

Fast scientific analysis

Upload an image or PDF, optionally crop to a region of interest, then place the baseline, optional axis tick, control, and treatment markers. Set the baseline value explicitly, then use a different axis tick only when you want value calibration.

Workflow

Compare mode is the original bar/segment workflow. Series mode lets you capture many points into one or more labeled traces and export raw/calibrated X/Y data.

Input

Example image: Huso et al., "Creatine supplementation influences substrate utilization at rest," Journal of Applied Physiology, 2002. Saving the current view makes future `Load Example` clicks restore the same image, markers, and values.

Y calibration still uses the baseline line and optional axis tick above. X calibration uses two point markers along the horizontal axis: origin and one tick.
Choose a series, left click to add points in order, then right click or use Finish Active Series to stop capturing. Existing series points can still be dragged.
