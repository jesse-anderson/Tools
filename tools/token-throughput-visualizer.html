Notes

This is an offline visualizer. Time to first token controls when visible output begins. Burstiness changes cadence without changing average rate. Share links carry trimmed state and do not require server storage.

Usage Disclaimer

  • This tool is an approximation and visualization aid, not a tokenizer, billing meter, or provider benchmark.
  • Throughput, cadence, and character density are modeled inputs. Real model output can differ because of tokenizer choice, server behavior, network latency, safety filters, and provider-side buffering.
  • Copied reports and share links are for communication convenience only and should not be treated as contractual performance evidence.
  • Custom sample text is stored locally in your browser and may be embedded in trimmed form in copied share links.
  • This tool is provided as-is, with no warranty. Verify important measurements against real logs or provider telemetry.
